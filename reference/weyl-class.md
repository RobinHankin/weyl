# Class “weyl”

The formal S4 class for weyls.

## Objects from the Class

Objects *can* be created by calls of the form `new("weyl", ...)` but
this is not encouraged. Use functions
[`weyl()`](https://robinhankin.github.io/weyl/reference/weyl.md) or
[`as.weyl()`](https://robinhankin.github.io/weyl/reference/weyl.md)
instead.
