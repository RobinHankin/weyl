# Generating elements for the first Weyl algebra

Variables `x` and `d` correspond to operator \\x\\ and \\\partial_x\\;
they are provided for convenience. These elements generate the
one-dimensional Weyl algebra.

## Usage

``` r
data(x_and_d)
```

## Details

Note that a similar system for multivariate Weyl algebras is not
desirable. We might want to consider the Weyl algebra generated by
\\\left\lbrace x,y,z,\partial_x,\partial_y,\partial_z\right\rbrace\\ and
correspondingly define R variables `x,y,z,dx,dy,dz`. But then variable
`x` is ambiguous: is it a member of the first Weyl algebra or the third?

## Examples

``` r
d
#> A member of the Weyl algebra:
#>  x d     val
#>  0 1  =    1
x
#> A member of the Weyl algebra:
#>  x d     val
#>  1 0  =    1

.[d,x]   # dx-xd==1
#> [1] 1

d^3 * x^4
#> A member of the Weyl algebra:
#>  x d     val
#>  4 3  =    1
#>  3 2  =   12
#>  2 1  =   36
#>  1 0  =   24

(1 - d*x*d) * (x^2 - d^3)
#> A member of the Weyl algebra:
#>  x d     val
#>  2 1  =   -5
#>  1 0  =   -4
#>  0 3  =   -1
#>  0 4  =    1
#>  2 0  =    1
#>  3 2  =   -1
#>  1 5  =    1
```
